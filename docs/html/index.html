DreamHacker

DreamHacker is a flexible tool which allows user to learn their sleep cycle, beats and other biological information about their sleep. It helps in manipulating their dreams for more comfortable sleep using sound stimulus.

Website interface enables user to select the music they want to play while they are sleeping.

Implementations

  • You can try out the implementations by following readMe or wiki.
    1. Then run it using a c++ compiler
  • Simulated data is used to prove the proof of concept.
  • Sleep Data like beats per minute is plotted using Dygraph and QT Application.
