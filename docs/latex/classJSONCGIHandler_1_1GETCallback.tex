\doxysubsection{Detailed Description}
GET callback handler which needs to be implemented by the main program. This needs to provide the JSON payload as a string either by using the simple \mbox{\hyperlink{classJSONCGIHandler_1_1JSONGenerator}{JSONGenerator}} or by an external library.
